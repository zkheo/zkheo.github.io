# 授权破解

复制服务器 ID，在存放 atlassian-agent.jar 的目录下执行命令，生成许可证：
邮箱 -m：随意
名称 -n：随意
访问地址 -o：你的 jira 访问地址
服务器 ID -s：刚刚复制的服务器 ID

将生成的许可证复制到页面，完成破解。
